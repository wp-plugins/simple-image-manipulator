Manipulate your wordpress images using slider and tickboxes.

== Description ==

<p><strong>Simple Image Manipulator</strong> is an simplest way that any one can manipulate images on wordpress. No need to use photoshop for adding effets or resizing images anymore because thats what this plugin does. This is the first version of the plugin. New versions will have more features, till than enjoy SIM Version 1.0.</p>

<p><strong>SIM Features</strong>
<ul>
<li>Resize images using Slider, yes thats right ;)</li>
<li>Add Grayscale</li>
<li>Add Sepia</li>
<li>Add Border</li>
<li>Add Border Color and size</li>
<li>Add Reflection</li>
<li>Auto upload to Media Library on save</li>
<li>Download Image to local machine</li>
</ul>
</p>


<p style="font-size:10px;">Limitations: Supported image types: gif,jpeg,png. Max image width 2000px;</p>

== Installation ==

1. Download plugin from the Wordpress directory.

2. Upload plugin using Add new plugin from plugin management.

3. Activate the plugin

4. Start using it from under Settings -> Simple Image Manipulator 



== Frequently Asked Questions ==

= What should I do after activating the plugin =

After activation. Go to Settings-> Find out Simple Image Manipulator and click on it. Then you will see all the images listed in a grid structure. These are the images from media library that you have uploaded. Just click on any image and start having fun with it.

= Where do I upload images =

Upload images using media library. Once uploaded it will list in SIM.

= Where does plugin saves the manipulated image =

Media Libray
